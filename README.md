# VIT Bhopal - 360° Campus Virtual Tour 🎓

Welcome to the **Project Exhibition 01** submission for Group-1! 

This project provides an immersive, lush green 360-degree virtual tour of the VIT Bhopal campus. Users can easily explore facilities, academic blocks, and campus life right from their screens using interactive panoramic views.

## Features ✨
- **Immersive 360° Viewing**: Explore the campus smoothly in full panoramic view.
- **VR Tour Integration**: Experience true immersion with stereoscopic VR views.
- **Custom Aesthetic**: A highly polished, modern, and interactive user interface.
